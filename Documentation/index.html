In this section, I tried to describe in detail how to set up a template for your restaurant if you are not familiar or cannot work with the code.

How to edit images

Template not include images which you see in demo!

  • To change the background images, you need to go to the images folder and the bg subfolder, upload your images there, which you will use as the background

    Now go to css folder and open main.css file, find the following sections:

    header
    oh
    gallery-header
    rv-header
    review

    Find the background-image selector in these sections and change the URL to the path to your image.
    example:
    background-image: url('../images/bg/image.png');

  • To change the dishes images in the menu section, also go to the images folder and upload the photos of the dishes to the menu subfolder.

    Now open your index.html file and find the following code blocks: 

                                    
    <div class="col-xl-4 col-md-6">
        <div class="menu_item animated fadeIn">
                    
            <div class="menu-image_container">
                <div class="menu-image_wrapper">
                    <div class="menu_image" style="background-image: url('images/menu/menu_image1.jpg');"></div>
                </div>
            </div>
                    
            <div class="content">
                <h4 class="menu-content_title">Pizza</h4>
                <span class="overlay"></span>
                <h4 class="menu-content_price">$25</h4>
            </div>
        </div>
    </div>

    As above, change the URL in the backgroung-image property to the path to your image.

  • To change the images in the gallery, go to the images folder and the gallery subfolder and upload your images there.

    Now open your index.html file and find the following code blocks: (there are 4 of them)

                                    
    <div class="gallery_item col-xl-7">
        <div class="padding_wrap one">
            <a class="gallery_image" href="images/gallery/gallery_image1.jpg" style="height: 350px;">
                <img src="images/gallery/gallery_image1.jpg" alt="Reload page!" width="635px" height="350px">
            </a>
        </div>
    </div>

    In each of the blocks in the href and src attributes specify the paths to your images.

  • To change a person's avatar in the reviews slider, go to the images folder and the avatar subfolder and upload images there (if they are not there, just leave it as it is).

    Now open index.html file and find the following code blocks: 

                                    
    <div class="col-xl-4">
        <div class="review_wrap">
            <img src="images/avatars/avatar.png" alt="Reload Page!" class="review_avatar">
                        
                <h5 class="review_title">John Doe</h5>
                        
                <div class="stars">
                    <i class="fas fa-star" id="star"></i>
                    <i class="fas fa-star" id="star"></i>
                    <i class="fas fa-star" id="star"></i>
                    <i class="fas fa-star" id="star"></i>
                    <i class="fas fa-star" id="star"></i>
                </div>
                        
                <h6 class="review_text">
                    Lorem ipsum dolor sit <br>
                    amet, consectetur <br>
                    adipiscing elit. Nam <br>
                    egestas elit a mi lacinia <br>
                    posuere. Fusce rhoncus 
                </h6>
        </div>
    </div>

    This is the code of the recall block, find the img tag and the scr attribute there, change the URL to the path to your image.

  • To change Instagram photos in footer, go to images folder and subfolder footer-instagram-images and upload photos there.

    Now open index.html file and find the following code blocks: 

                                    
    <div class="insta_photo-content">
        <div class="insta_photo">
            <img src="images/footer-instagram-images/insta_img1.jpg" alt="Reload page!">
        </div>
                
        <div class="insta_photo">
            <img src="images/footer-instagram-images/insta_img2.jpg" alt="Reload page!">
        </div>
                
        <div class="insta_photo">
            <img src="images/footer-instagram-images/insta_img3.jpg" alt="Reload page!">
        </div>
    </div>

    Find the img tag and the scr attribute there, change the URL to the paths to your images.

  • To change the logo, go to the images folder and the logos subfolder and upload the logos named logo.svg and footer_logo.svg there.

    If your logo is in another format like PNG or JPG, use an online converter.

Customizing reservation form

In order for the booking form to work, you need 2 email accounts!
One account submits the application, and the other receives.

Attention! This form only works on hosting or local server.

  • Open mail.php file.

    You will see the this: 

                                    
    <?php 

    require_once('phpmailer/PHPMailerAutoload.php');
    $mail = new PHPMailer;
    $mail->CharSet = 'utf-8';
                                    
    $time = $_POST['time'];
    $datepicker = $_POST['datepicker'];
    $persons = $_POST['persons'];
    $name = $_POST['name'];
    $phone = $_POST['phone'];
    $regard = $_POST['regard'];
                                    
    //$mail->SMTPDebug = 3;                         // Enable verbose debug output
                                    
    $mail->isSMTP();                                      // Set mailer to use SMTP
    $mail->Host = 'smtp.mail.ru';  																							// Specify main and backup SMTP servers
    $mail->SMTPAuth = true;                               // Enable SMTP authentication
    $mail->Username = 'youmail@mail.com'; // Your login from the email from which letters will be sent
    $mail->Password = 'password'; // Your password from the email from which letters will be sent
    $mail->SMTPSecure = 'ssl';                            // Enable TLS encryption, `ssl` also accepted
    $mail->Port = 465; // TCP port to connect to / this port may be different from other providers
                                    
    $mail->setFrom('youmail@mail.com'); // From whom will the letter go?
    $mail->addAddress('mail@domain.ru');     // Who will receive the letter?
    //$mail->addAddress('ellen@example.com');               // Name is optional
    //$mail->addReplyTo('info@example.com', 'Information');
    //$mail->addCC('cc@example.com');
    //$mail->addBCC('bcc@example.com');
    //$mail->addAttachment('/var/tmp/file.tar.gz');         // Add attachments
    //$mail->addAttachment('/tmp/image.jpg', 'new.jpg');    // Optional name
    $mail->isHTML(true);                                  // Set email format to HTML
                                    
    $mail->Subject = 'NEW RESERVATION REQUEST';
    $mail->Body    = 'time: ' .$time. '<br>date: ' .$datepicker. '<br>persons: ' .$persons. '<br>name: ' .$name. '<br>phone: ' .$phone. '<br>regard: ' .$regard;
    $mail->AltBody = '';
                                    
    if(!$mail->send()) {
        header('location: error.html');
    } else {
        header('location: thank-you.html');
    }
    ?>

    We will only work with this part of the code:

                                    
    $mail->isSMTP();                                      // Set mailer to use SMTP
    $mail->Host = 'smtp.mail.ru';  																							// Specify main and backup SMTP servers
    $mail->SMTPAuth = true;       // Enable SMTP authentication
    $mail->Username = 'youmail@mail.com'; // Your login from the email from which letters will be sent
    $mail->Password = 'password'; // Your password from the email from which letters will be sent
    $mail->SMTPSecure = 'ssl';    // Enable TLS encryption, `ssl` also accepted
    $mail->Port = 465; // TCP port to connect to / this port may be different from other providers
                                    
    $mail->setFrom('youmail@mail.com'); // From whom will the letter go?
    $mail->addAddress('mail@domain.ru');     // Who will receive the letter?

    Enter the SMTP of your mail from which letters will be sent (the list of SMTP for different mailboxes is given below).

    • Gmail: smtp.gmail.com
    • Yahoo: smtp.mail.yahoo.com
    • Aol: smtp.aol.com
    • iCloud: smtp.mail.me.com
    • Mail.ru: smtp.mail.ru
    • Yandex.Mail: smtp.yandex.ru
    • port: 465
    • port: 465 or 587
    • port: 587
    • port: 587
    • port: 465
    • port: 465

    Now enter your mail SMTP here: (I have mail.ru for example) 

    
    $mail->Host = 'smtp.mail.ru';

    Now we need to specify the login and password of the mail from which messages will be sent: 

                                    
    $mail->Username = 'youmail@mail.com'; // Your login from the email from which letters will be sent
    $mail->Password = 'password'; // Your password from the email from which letters will be sent

    The next step we need to specify the port, the list of ports for each mailbox is given above 

    
    $mail->Port = 465; // TCP port to connect to / this port may be different from other providers

    I have port 465.

    Now enter the mail from which letters will be sent (we entered it above) and the mail to which letters will come (enter the main mail)
